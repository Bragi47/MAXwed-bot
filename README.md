# MAXwed Bot

**Telegram-бот для быстрого доступа к веб-версии MAX.**

При нажатии кнопки сайт открывается внутри Telegram как Mini App.

## Быстрый старт
```bash

# 1. Клонируй репозиторий
git clone https://github.com/Bragi47/MAXwed-bot.git
cd MAXwed-bot

# 2. Запусти менеджер — сам запросит токен, настроит и запустит бота
chmod +x maxwed.sh
./maxwed

## Команды бота

| Команда    | Описание                              |
|------------|---------------------------------------|
| `/start`   | Приветствие и кнопка «Открыть MAX»    |
| `/help`    | Список команд                         |
| `/about`   | Информация о боте                     |

## Менеджер `maxwed.sh`

Интерактивное меню для управления:

| Пункт | Действие                           |
|-------|------------------------------------|
| 1     | Полная установка (первичная)       |
| 2     | Запустить бота                     |
| 3     | Остановить бота                    |
| 4     | Перезапустить (пересобрать образ)  |
| 5     | Показать логи                      |
| 6     | Статус                             |
| 7     | Обновить (git pull + rebuild)      |
| 8     | Сменить токен                      |
| 9     | Полностью удалить                  |
| 10    | Выход                              |

## Docker Compose
```bash

# Сборка и запуск
docker compose up -d --build

# Логи
docker compose logs -f

# Остановка
docker compose down
```

## Безопасность

- Токен шифруется Fernet (`token.enc`), ключ (`key.bin`) — только на сервере
- `.env`, `key.bin`, `token.enc` исключены из репозитория (`.gitignore`)
- Rate limiter (макс. 10 запросов/мин с одного пользователя)
- HTML-escape всех user input
- Graceful shutdown по SIGTERM/SIGINT
- Контейнер работает от непривилегированного пользователя

## Технологии

- Python 3.12+
- aiogram 3.x
- Docker
