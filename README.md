# MAXwed Bot

**Telegram-бот для быстрого доступа к веб-версии MAX.**

При нажатии кнопки сайт открывается внутри Telegram как Mini App.

## Быстрый старт

```bash
# 1. Клонируй репозиторий
git clone https://github.com/Bragi47/MAXwed-bot.git
cd MAXwed-bot

# 2. Запусти менеджер — сам запросит токен, настроит и запустит бота
chmod +x maxwed.sh
./maxwed.sh
```

После первого запуска `./maxwed.sh` покажет меню для управления ботом.

## Команды

| Команда    | Описание                              |
|------------|---------------------------------------|
| `/start`   | Приветствие и кнопка «Открыть MAX»    |
| `/help`    | Список команд                         |
| `/about`   | Информация о боте                     |

## Безопасность

Токен хранится в зашифрованном виде (`token.enc`). Ключ расшифровки (`key.bin`) и файл `.env` **не попадают** в репозиторий (исключены через `.gitignore`).

При каждом новом клонировании запускай `python setup_encrypt.py` заново.

## Технологии
